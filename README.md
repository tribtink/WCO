# World Citizens Organization

A neutral civic substrate for global coordination.

This repository contains the early scaffolding for the WCO runtime:
- minimal MV3 browser extension shell
- substrate initialization loop
- placeholder civic object ecology
- pre-launch Civic Reciprocity License

Nothing here is final. Everything is subject to change.

## Status

Pre-launch.  
Only the extension frame and basic substrate files are published.  
Civic dynamics, schemas, and meaning-kernel internals remain unpublished.
