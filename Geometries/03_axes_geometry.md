# Axes of the geometry

The coordination substrate is modeled as a **high‑dimensional geometry** with several key axes.

> “Components: Ontology, topology, capability. Think of this as: what exists, how it’s wired, and what it can do.”   

We group them into:

- structural axis  
- runtime axis  
- scope axis  
- context axis  
- temporal axis  

## Structural axis

The structural axis describes:

- **Ontology:** what entities and relations exist.  
- **Topology:** how entities are connected (graphs, layers, multiplex networks).  
- **Capability:** what actions and transformations are possible.  

Formally, we can think of a structural configuration as:

- a set of nodes (agents, institutions, resources)  
- a set of edges (relations, channels, dependencies)  
- a capability mapping (who can do what to whom/what)  

## Runtime axis

The runtime axis captures:

- **State:** current configuration of variables (trust levels, power distributions, institutional status, etc.).  
- **Dynamics:** update rules and processes (learning, adaptation, escalation, reform).  
- **Outcomes:** realized trajectories and payoffs over time.  

This is where we model:

- feedback loops  
- tipping points  
- path dependence  

## Scope axis

The scope axis ranges from:

- **individual**  
- **group**  
- **organization**  
- **city/region**  
- **state**  
- **civilization**  

The same primitives and structures apply at each level; only the scale and aggregation differ.

## Context axis

The context axis spans:

- **cooperative** ↔ **adversarial**  

This is not a binary switch but a **continuous and possibly heterogeneous field**:

- some subsystems may be cooperative while others are adversarial  
- context can shift over time due to shocks, betrayals, or successful trust‑building  

## Temporal axis

The temporal axis ranges from:

- **immediate** (tactical, short‑term)  
- **strategic** (medium‑term)  
- **civilizational** (long‑term, structural)  

Different invariants and dynamics dominate at different temporal scales.

## State–context space

A **coordination system** at any moment is a point in this geometry:

- structural configuration  
- runtime state  
- scope position  
- context value  
- temporal horizon  

Trajectories through this space correspond to **histories** of peace, war, cooperation, breakdown, and transformation.
